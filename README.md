## 🚀 Featured Projects

* **[Fotobudka](https://fotobudka.club)** A SaaS platform built for events like weddings. Guests scan a QR code to upload photos directly to a centralized dashboard for the organizers. 
    *Tech: AstroJS, TypeScript, Stripe, Prisma ORM, Postgres, Kubernetes*
* **[Pi Pico Foot Pedal](https://github.com/MonkaKokosowa/foot-pedal)**
    A custom, programmable USB HID macro pedal built from scratch using mechanical keyboard switches and a Raspberry Pi Pico. 
    *Tech: CircuitPython, Rust, OpenDeck, USB HID*
* **[LiveMPK](https://github.com/MonkaKokosowa/tabler-c)**
    A live bus tracking dashboard for Raspberry Pi. Built by reverse-engineering legacy public transit systems to display real-time schedules on any monitor.
    *Tech: C, LVGL, Makefile*
* **[Pi Temperature Monitoring](https://github.com/MonkaKokosowa/PiMonitoring)**
    A custom hardware/software monitoring solution for my k3s Raspberry Pi cluster, utilizing an Arduino UNO R4 and an I2C LiquidCrystal display to track thermals.
    *Tech: Arduino/C, Python, Systemd*
